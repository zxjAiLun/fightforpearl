# 战斗系统修复 Spec

## Why

修复当前战斗系统中发现的多个问题：
1. 怪物行动错误增加战绩点
2. 怪物伤害不应暴击
3. 能量显示格式需要调整
4. 行动值计算需要修正

## What Changes

### 1. 战绩点仅玩家可用

**问题**：怪物行动也在增加BP
**修复**：战绩点（团队共享）仅对玩家角色有效，怪物不应使用或增加战绩点

### 2. 怪物不暴击

**问题**：怪物伤害也会暴击
**修复**：敌人（is_enemy=True）不进行暴击判定，暴击率固定为0

### 3. 能量显示格式

**修复**：不使用能量条，直接显示数值如 `能量:30/120`

### 4. 行动值计算修正

**当前问题**：133.4速度的角色在第一轮行动两次
**原因**：第一轮行动值应该为150，而非速度计算值

**修正规则**：
- 第一轮行动值 = 150（简化计算）
- 后续回合行动值 = 100

**计算公式**：
```
第一轮：初始行动值 = 150 / 速度
第二轮+：初始行动值 = 100 / 速度
```

### 5. 测试敌人血量

**规则**：测试中涉及多个enemy时，将怪物血量调低至 `hp_units=1`

## Impact

- Affected code:
  - `src/game/battle.py` - 战绩点逻辑
  - `src/game/damage.py` - 暴击判定
  - `src/game/tui.py` - 能量显示
  - `src/game/models.py` - 行动值初始化
  - `tests/` - 测试数据

## MODIFIED Requirements

### Requirement: 战绩点

战绩点系统仅对玩家角色生效，怪物角色不参与战绩点系统。

### Requirement: 暴击判定

敌人不进行暴击判定，暴击率 = 0。

### Requirement: 行动值初始化

第一轮行动值 = 150，后续 = 100。

### Requirement: 能量显示

直接显示数值，不使用能量条。
