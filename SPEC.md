## 1. 核心战斗公式（已验证）

### 1.1 伤害公式

```
FinalDMG = ATK × Multiplier × (1 - DEF_RED) × (1 + DMG%) × CRIT × (1 + VULN) + 固定伤害
```

**防御减伤公式（崩铁标准）**：
```
DEF_RED = DEF / (DEF + 200 + 10 × Level)
```

角色攻击怪物时（玩家视角）：
```
DEF_RED = 100 / (100 + Enemy_Level × 20 × (1 - Pen))
```

怪物攻击角色时：
```
DEF_RED = (10 × Attacker_Level + 200) / (100 × Attacker_Level + 200 + DEF)
```

**暴击判定**：rand < crit_rate 时暴击，伤害 × crit_dmg

**易伤区**：每层易伤增加伤害

### 1.2 行动值系统（Action Value）

```
行动值基准 = 10000（AV_BASE）
行动值 = AV_BASE / 速度

第一轮 = 10000 / 速度
后续 = 10000 / 速度
```

行动后重置：+10000 AV

**拉条/推条**：
- 每1%提前 = 100 AV
- 每1%延后 = 100 AV

### 1.3 能量系统

- 能量上限：120（可配置）
- 初始能量：能量上限 / 2 = 60
- 普攻回复：20能量
- 战技回复：30能量
- 击杀回能：10能量
- 受击回能：10能量
- 大招释放：消耗全部能量，能量重置为0
- 能量恢复效率：影响所有能量回复

### 1.4 战绩点系统（Battle Points / Break Points）

- 战绩点上限：6点（暂定）
- 初始战绩点：3点
- **战技消耗**：1点战绩点
- **普攻回复**：1点战绩点
- 只有玩家增加战绩点，怪物不增加

### 1.5 弱点击破系统

**击破触发条件**：
1. 目标韧性条 ≤ 0
2. 攻击属性在目标弱点列表中

**击破伤害**：
```
BreakDMG = Level × 10 × BreakCoeff × ToughnessCoeff × (1 + BreakEFF)
```

**击破效果类型**：

| 元素 | 效果 | 持续 |
|------|------|------|
| 物理 | 裂伤：HP%持续伤害 | 2回合 |
| 火 | 灼烧：火属性DOT | 2回合 |
| 冰 | 冻结：无法行动 | 2回合 |
| 雷 | 触电：雷属性DOT | 2回合 |
| 风 | 风化：可叠加DOT | 2回合 |
| 量子 | 纠缠：行动延后+额外伤害 | 2回合 |
| 虚数 | 禁锢：行动延后+减速 | 2回合 |

### 1.6 敌人血量公式

```
Enemy_HP = heart_count × HEART_BASE × difficulty_coeff × variant_coeff

HEART_BASE = 21997.729（90级标准血量）
difficulty_coeff: 默认=1.0, 混沌=1.226
```

## 4. 当前问题清单

### Issue #1: 战技消耗逻辑错误
**描述**：当前战技消耗战绩点，但代码中没有检查战绩点是否足够
**影响**：战技可以无限使用
**修复**：在 skill.py 中添加战绩点消耗检查

### Issue #2: 行动值常量不统一
**描述**：config 中 FIRST_ROUND_AV=150, SUBSEQUENT_AV=100，但 ACTION_COST=10000
**影响**：计算混乱
**修复**：统一为10000基准

### Issue #3: 伤害数值不符合崩铁
**描述**：角色攻击力2000，怪物血量219977，但崩铁中这些数值更小
**影响**：数值不平衡，不符合原版体验
**修复**：调整数值到合理范围
