# 战斗与数值设计文档

> 记录核心战斗机制、数值框架与平衡设计思路。

---

## 1. AV 回合制系统

### 行动值（Action Value）

每个角色和敌人都有 **AV**，速度决定 AV 消耗速度：

```
AV 消耗 = 10000 / 速度
```

速度越高，AV 消耗越快，越快获得下一次行动机会。

### 回合流程

1. 所有单位按当前 AV 排序，AV 最低者先行动
2. 行动后 AV 重置为 10000
3. 所有单位 AV 减去当前最低 AV，保持游戏推进
4. 循环直到战斗结束

### 追击机制

- 击杀敌人后，触发**追击**：随机己方单位追加一次行动
- 追击不消耗额外回合，直接插入行动序列

## 2. 伤害计算

### 基础公式

```
damage = baseDamage × (1 + damageBonus) × (1 - damageReduction) + flatBonus
```

### 暴击

```
critDamage = baseDamage × critMultiplier
critChance 受属性和光锥影响
```

### 属性伤害

每种属性（物理、火、冰、雷、风、虚数、量子）有独立增伤区间，多区间叠加。

## 3. 角色系统

### 命途

| 命途 | 定位 | 代表能力 |
|------|------|----------|
| 毁灭 | 均衡输出 | 高持续伤害 |
| 巡猎 | 单体爆发 | 追击触发率 |
| 智识 | AOE 输出 | 群体伤害 |
| 同谐 | 辅助增益 | 增伤/拉条 |
| 虚无 | 负面效果 | debuff 叠加 |
| 存护 | 防御 | 护盾/减伤 |
| 丰饶 | 治疗 | 持续回复 |

### 技能类型

- **普攻**：基础伤害，触发追击
- **战技**：消耗能量，造成额外效果
- **终结技**：消耗能量，造成高额伤害
- **被动**：条件触发，提供增益或反击

---

## 4. 光锥系统

光锥提供：
- 基础属性（生命、攻击、防御）
- 武器特效（增伤、暴击、能量回复等）

光锥效果与角色命途存在关联，某些光锥对特定命途角色有额外加成。

---

## 5. 遗器系统

遗器提供：
- 主词条：固定属性（攻击%、生命%等）
- 副词条：随机属性（暴击率、速度等）
- 套装效果：2件/4件触发额外效果

---

## 6. 模拟宇宙 roguelike

### 地图生成

每次探索随机生成节点网络：
- **战斗节点**：击败敌人获得祝福/奇物
- **奖励节点**：直接获得增益
- **商店节点**：消耗星芒购买祝福
- **事件节点**：随机事件（增益/减益）
- **首领节点**：击败 boss 获得丰厚奖励

### 祝福系统

祝福分多种类型：
- **回响**：基础属性提升
- **触发**：特定条件下触发额外效果
- **赋予**：获得新能力

### 难度设计

```
敌人等级 = 基础等级 + (难度等级 × 0.5)
敌人属性 = 基础属性 × (1 + 难度等级 × 0.1)
```

---

## 7. 敌人 AI

### 行为模式

- **攻击型**：优先攻击己方脆皮目标
- **防守型**：优先叠加增益/护盾
- **集火型**：锁定单一目标击杀
- **平衡型**：根据场面情况选择目标

### 技能优先级

敌人释放技能优先级基于：
1. 终结技（能量充足时优先）
2. 战技（资源允许时）
3. 普攻（默认填充）

---

## 8. 能量系统

- **能量上限**：角色 120（部分角色有差异）
- **能量获取**：受击+20、击杀+30、回合开始+5
- **终结技**：消耗 120 能量释放

---

## 9. 关键数值对照

| 指标 | 数值 |
|------|------|
| 能量上限 | 120 |
| 速度基准 | 100 |
| 暴击率上限 | 100% |
| 暴击伤害基准 | 150% |
| 伤害加成上限 | ~500%+ |
| 护盾计算 | 独立于生命池 |
