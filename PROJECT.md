# Fight for Pearl — 项目进度总结 v0.5.0

## 项目概述

**Fight for Pearl** 是一个回合制战斗框架，灵感来源于崩坏星穹铁道的战斗系统。当前版本 **v0.5.0**。

## 项目目标

**复刻崩坏星穹铁道回合制战斗逻辑**
- 优先级：战斗逻辑 > GUI > 养成系统（暂不实现）
- 养成逻辑目前用手填数据

### ✅ 追加攻击触发器（v0.5.0新增）

#### 触发条件类型
| 条件 | 说明 |
|------|------|
| `TARGET_HP_BELOW` | 目标HP低于阈值 |
| `TARGET_HP_ABOVE` | 目标HP高于阈值 |
| `TARGET_IS_WEAKENED` | 目标处于弱化状态 |
| `CASTER_HP_BELOW` | 自身HP低于阈值 |
| `CASTER_HP_ABOVE` | 自身HP高于阈值 |
| `AFTER_BASIC` | 普攻后必然触发 |
| `AFTER_SPECIAL` | 战技后必然触发 |
| `KILL` | 击杀后触发 |
| `RANDOM` | 随机概率触发 |

#### 目标范围
- `single` — 只攻击触发追加攻击的目标
- `aoe` — 攻击所有满足条件的敌人
- `trigger` — 只攻击触发目标

## 运行方式

```bash

source .venv/bin/activate

# 运行GUI

# 运行TUI
```
