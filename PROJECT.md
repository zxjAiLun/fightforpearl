# Fight for Pearl — 项目进度总结 v0.4.0

## 项目概述

**Fight for Pearl** 是一个回合制战斗框架，灵感来源于崩坏星穹铁道的战斗系统。当前版本 **v0.4.0**。

## 项目目标

**复刻崩坏星穹铁道回合制战斗逻辑**
- 优先级：战斗逻辑 > GUI > 养成系统（暂不实现）
- 养成逻辑目前用手填数据

---

## 已完成功能

### ✅ 战斗引擎核心
- [x] 回合制战斗流程
- [x] 行动值（Action Value）系统 — 基于速度的行动排序
- [x] 回合标记（Round Marker）机制
- [x] Step Back功能（回退到上一个行动）
- [x] DOT触发和结算
- [x] 行动角色选择bug修复（max→min）

### ✅ 技能系统
- [x] 普通攻击（Basic Attack）— 不消耗战绩点，回复1点战绩点
- [x] 战技（Special Skill）— 消耗1点战绩点，回复能量
- [x] 大招（Ultimate）— 消耗全部能量，需能量满
- [x] 追击（Follow-Up）— 满足条件后触发的额外攻击
- [x] 被动技能触发系统

### ✅ 技能类型（v0.4.0新增）
- [x] **AOE技能** — 攻击多个目标（target_count > 1 或 target_count = -1）
- [x] **弹射技能** — 弹射n次，每次弹射伤害递减（ricochet_count, ricochet_decay）
- [x] **扩散技能** — 主目标全额伤害，扩散目标受spread_multiplier影响

### ✅ 能量与战绩点系统
- [x] 能量上限：120点
- [x] 初始能量：60点
- [x] 普攻回复：20能量
- [x] 战技回复：30能量
- [x] 战绩点：团队共享，上限6点
- [x] **战技消耗1点战绩点**

### ✅ 效果命中/抵抗系统
- [x] 命中率公式：`效果命中 / (效果命中 + 效果抵抗 + 23)`
- [x] DOT和控制效果需要判定命中率
- [x] 裂伤（HP%伤害）无需判定，始终命中

### ✅ 敌人AI
- [x] 优先使用大招（能量满时）
- [x] 有多个目标时优先使用AOE技能
- [x] 优先攻击血量最低的目标

### ✅ 伤害计算
- [x] 崩铁风格伤害公式
- [x] 防御减伤公式（两种）
- [x] 暴击判定（敌人不暴击）
- [x] 易伤区叠加
- [x] 增伤区叠加
- [x] 击破伤害计算

### ✅ 击破系统
- [x] 7种元素对应7种击破效果

### ✅ GUI/TUI界面
- [x] Pygame图形界面
- [x] 命令行文字界面
- [x] 行动条显示
- [x] 战斗日志面板
- [x] 技能信息面板（显示当前可用水面）
- [x] Step/Back/Restart/Export 按钮

## 运行方式

```bash
# 进入虚拟环境
source .venv/bin/activate

# 运行GUI
python -m src.game.gui

# 运行TUI（默认4人队伍 vs 2敌人）
python -m src.game.tui

# 运行TUI（显示完整日志）
python -m src.game.tui --full

# 运行TUI（自定义队伍）
python -m src.game.tui --team "星,银狼,姬子,布洛妮娅"

# 运行TUI（指定敌人数量）
python -m src.game.tui --enemy 3

# 运行测试
python -m pytest tests/ -v
```
