### 阶段 3：属性系统重构 + 被动技能 ✅
- [x] **移除元素克制系统**（崩铁无元素克制，元素仅影响弱点击破）
- [x] **分离基础属性 vs 百分比属性**
  - `base_atk / base_def / base_spd / base_max_hp` — 基础值
  - `atk_pct / def_pct / spd_pct / hp_pct` — 百分比加成
  - `atk_flat / def_flat / hp_flat` — 固定值加成
  - 最终值 = `(基础 + 固定) × (1 + 百分比)`
- [x] **修正伤害公式分区**
  - 增伤区 DMG% = 1 + sum(所有伤害加成)，同类相加
  - 易伤区 VULN = 1 + sum(所有易伤)，同类相加
  - 暴击区 CRIT_MULT = 暴击? CRIT_DMG : 1
  - 击破区 BREAK（独立乘法）
  - 公式：`FinalDMG = ATK × Multiplier × (1-DEF_RED) × DMG% × CRIT × BREAK × VULN`
- [x] **弱点击破系统框架**（BreakEffectType：裂伤/灼烧/冻结/触电/风化/纠缠/禁锢）
- [x] **被动技能系统**
  - 被动1（战技·增伤）：释放战技后，造成伤害+30%，持续2回合
  - 被动2（大招·加攻）：释放大招后，攻击力+30%，持续2回合
- [x] 效果系统 Effect（应用/移除属性加成，回合递减）
